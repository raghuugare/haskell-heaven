Functions

Let us consider some familiar and not-so-familiar examples involving functions.

Remember Mathematics? \(ax^{2}+ \ldots \) ??

Consider the famous Quadratic equation :

\( a x^{2} + b x + c = 0 \)

The above equation states that some function of the independent variable \(x\) is equal to the expression \( {\mbox ax^{2} + bx + c }\) and that it evaluates to zero for all (input) values of \(x\).
Symbolically, we can write this as follows:

\(f(x) = ax^{2} + bx + c = 0 \)

Ok, how about Physics ?

Or...do you remember the equation from the Physics of Motion describing the final velocity \(v\) of an object with an initial velocity of \(u\) units, moving under a constant acceleration of \( a \) units after a time \( t \) ??

\( v = u + at \)

Yes!
There's a function hidden here too.
Given all things, the final velocity \( v \) is a function of time, and can be written as:

\(v(t) = u + at\)

What then is a function?

A function \(f\) is simply a mapping from a set (called the domain) to another set, which is usually called the function's range.
