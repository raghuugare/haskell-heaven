Types

Types are Haskell's best approximations of the mathematical notion of Sets. In fact, Haskell takes the notion of types to a deeper level, associating even functions with their own 'types', we shall see soon now.

What is a type?

At its simplest, a type can be thought of as a set of related values sharing some common traits.

Some simple examples of types are Bool which is a set of only 2 values viz., \(\{True, False\} \). Other common types are Int & Integer which represent the set of all integers which is shown in math by the symbol \(\mathbb{Z}\).

In Haskell, a function \(f\) that maps an argument of type Int to a return value of type Int is depicted as below:

\(f :: Int \rightarrow Int\)

In reality, however, we do have some issues. Although the set of integers is essentially infinite, the set of integers we have on computers is actually finite. You can think of this in terms of a concept in mathematics you might have come across in Calculus, particularly while dealing with the infinite set of the so called Real Numbers denoted by \( \mathbb{R} \) – called the interval of a set.

An interval of a set is a subset of the set such that it contains atleast 2 members and contains all members of the set between any 2 elements belonging to the interval (i.e., no member of the set is left out in the interval).

E.g., the set of all integers such that \( x > 8 \) is an interval. But the set of all non-zero integers is not an interval since it leaves out the element zero (\(0\)). Clearly an interval could be finite or infinite. Also, if the end-points of the interval belong to it, the interval is said to be closed; else it is called an open interval. Of course, as you must have guessed, an interval can be closed or opern at either end-points. Below are some examples in set notation.

Examples of Finite Intervals in \( \mathbb{R} \)
Finite Intervals
Interval Set Notation
\((a, b)\) \(\{ x | a < x < b\} \)
\([a, b]\) \(\{ x | a \leq x \leq b\} \)
\([a, b)\) \(\{ x | a \leq x < b\} \)
\((a, b]\) \(\{ x | a < x \leq b\} \)

Examples of Infinite Intervals in \( \mathbb{R} \)
Infinite Intervals
Interval Set Notation
\((a, \infty ) \) \(\{ x | x > a \} \)
\([a, \infty ) \) \(\{ x | x \geq a \} \)
\((-\infty, b ) \) \(\{ x | x < b \} \)
\((-\infty, b ] \) \(\{ x | x \leq b \} \)
\((-\infty, \infty ) \) \( \mathbb{R} \) (the set of real numbers!)

In general, when we see something of the form \(v :: T\), it means a value \(v\) ``has type'' \(T\).
