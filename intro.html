Hello World

How about the "typical" program that prints "Hello World"?
Well, here goes the code of such a program called Hello.hs: 

				
						-- Hello.hs
						main :: IO ()
						main = print "Hello World"

Note: Do not let the program-snippet syntax fool you about the I/O system of Haskell being "typical" in any way--especially if you are from an imperative programming language like Java, C++, etc.

Haskell is one of the rare languages with an almost maniacal obsession about the concept of purity. In a programming language, being pure means that all side-effects are modelled explicitly, rather than being present in an invisible, unpredictable way.

Please don't worry. We have a lot to speak about the features of Haskell in this space.

So brace yourselves. You're in for a magical journey into Haskell Heaven!!
